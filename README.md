# Zaris - Premium Leather Wallets

A complete e-commerce website for Zaris leather wallets with admin panel and essential features.

## Features

### Customer-Facing Website
- 🛍️ Product catalog with images and descriptions
- 🔍 Search functionality
- 🏷️ Product filters and sorting
- 🛒 Shopping cart
- 📱 WhatsApp order integration
- 📋 Complete checkout form (COD & Online Payment)
- ❓ FAQ section
- 📄 Information modals (Shipping, Returns, Care, Privacy, Terms)
- 💱 Multi-currency support (INR default)
- 📱 Fully mobile responsive

### Admin Panel
- 🔐 Secure login (admin/zaris123)
- ➕ Add/Edit/Delete products
- 📸 Direct image upload (base64 storage)
- 📊 Dashboard with statistics
- 📦 Order management
- 👥 Customer management
- 📈 Analytics
- ⚙️ Settings (currency, contact info, etc.)

## Tech Stack
- HTML5, CSS3, JavaScript (Vanilla JS)
- LocalStorage for data persistence
- No backend required - runs entirely client-side

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser for the store
3. Open `admin.html` in your browser for admin panel
4. Default admin credentials: `admin` / `zaris123`
